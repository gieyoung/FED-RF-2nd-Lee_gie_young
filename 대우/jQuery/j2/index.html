[jQuery 애니메이션 관련 메서드]

  1. $("대상")/.fadeIn(); : 숨겨져있던 대상이 서서히 보임
  2. $("대상")/.fadeOut(); :대상이 서서히 사라짐
  3. $("대상")/.fadeToggle(); : 숨김-나타내기-숨김
  4. $("대상")/.slideUp(); : 선택된 대상의 높이를 0으로 설정
  5. $("대상")/.slideDown(); : 대상의 높이를 설정된 높이만큼 아래로 애니메이션
  6. $("대상")/.slideToggle(); : 클릭-높이0-또클릭-높이 설정값으로
