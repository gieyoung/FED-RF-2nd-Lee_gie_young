jQuery란?

제이쿼리는 자바스크립트의 생산성을 향상시켜주는 자바스크립트의 필요한 함수를 모아둔 곳. 원하는 함수를 찾아서 사용하는 방식

jQuery특징?

  1. 앨리먼트(요소/마크업태그)를 선택하는 강력한 기능 제공
  2. 선택된 엘리먼트들을 제어할 수 있는 다양한 함수
  3. 제공 자바스크립트를 라이브러리화 함

jQuery문법?

  1. $("대상").메소드();
  2. jQuery("대상").메소드();

jquery랩퍼(Wrapper)?

  1. $(document).ready(function(){});
  2. $(function(){});

jQuery부호($)의 역할

*괄호안에 넣을 요소를 선택할 때 쓴다.

jQuery 부호($)에 들어가는 이름들?

  1. CSS셀렉터
  2. HTML요소
  3. 자바스크립트 객체

jQuery문서 연결방법

  1. jQuery문서 연결방법: 직접연결시 제이쿼리 공식사이트에서 라이브러리를 다운받아서 직접 문서에 연결하여 서비스 함
  2. CDN주소로 연결시:서비스를 제공하는 업체로부터 링크주소를 사용해서 문서에 연결하여 서비스 함
